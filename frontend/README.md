# Poornima CMS - Frontend Client

This is the Next.js frontend client for the Poornima Complaint Management System.

For complete local setup instructions, environment variable configurations, and backend integration guidelines, please refer to the main [Root README](../README.md).

## Local Development Quickstart

1. Copy env configuration:
   ```bash
   cp .env.local.example .env.local
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the Next.js dev server:
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` to view the application.
